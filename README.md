# 금은별장 가상 투어

금은별장의 전체 공간을 웹/모바일에서 둘러볼 수 있는 인터랙티브 가상 투어 웹사이트입니다.

일반 카메라로 촬영한 사진을 사용하여 진입로부터 모든 객실까지 자연스럽게 연결된 동선을 체험할 수 있습니다.

## 🚀 사용 방법

### 1. 사진 준비

각 공간마다 `main.jpg` 파일로 사진을 넣어주세요:

**권장 사항:**
- 가로로 촬영한 사진 사용 (16:9 비율 권장)
- 해상도: 최소 1920x1080px 이상
- 파일 크기: 500KB~2MB (웹 최적화)
- 일반 카메라나 스마트폰으로 촬영한 사진 사용 가능

### 2. 웹사이트 실행

#### 방법 1: 로컬 서버 사용 (권장)
```bash
# Python이 설치되어 있다면
python -m http.server 8000

# 또는 Node.js의 http-server 사용
npx http-server
```

브라우저에서 `http://localhost:8000` 접속

#### 방법 2: Live Server (VS Code)
VS Code에서 `index.html` 파일을 우클릭 → "Open with Live Server"

#### 방법 3: 직접 열기
`index.html` 파일을 더블클릭하여 브라우저에서 열기

## 🎯 주요 기능

### 1. **자연스러운 동선**
- 진입로부터 시작하여 실제 동선대로 이동
- 각 공간에서 연결된 공간으로 자연스럽게 이동

### 2. **핫스팟 시스템**
- 사진 위에 표시되는 버튼을 클릭하여 이동
- 직관적인 아이콘과 텍스트로 목적지 표시

### 3. **카테고리별 메뉴**
- 진입 구역, 주차 구역, 외부 시설, 1층, 2층 등으로 분류
- 우측 상단 메뉴(☰)에서 원하는 공간으로 바로 이동

### 4. **이미지 뷰어 기능**
- 줌 인/아웃 (1배 ~ 3배)
- 한 공간에 여러 장의 사진 추가 가능
- 좌우 화살표 또는 스와이프로 사진 전환

### 5. **모바일 최적화**
- 반응형 디자인
- 터치 제스처 지원
- 스크롤 가능한 메뉴

## 🛠️ 사진 추가/변경하는 방법

### 기본 사진 넣기

각 폴더에 `main.jpg` 파일로 사진을 넣으세요:

```bash
images/
  entrance-road/entrance-road.jpg      # 진입로 사진
  resort-entrance-1/resort-entrance-1.jpg  # 금은별장 진입로1 사진
  ...
```

### 한 공간에 여러 장의 사진 추가

`js/tour.js` 파일을 열어서 해당 공간의 `photos` 배열에 사진 추가:

```javascript
'living-room': {
    name: '거실',
    emoji: '🛋️',
    photos: [
        'images/living-room/main.jpg',
        'images/living-room/view1.jpg',  // 추가
        'images/living-room/view2.jpg'   // 추가
    ],
    // ...
}
```

### 핫스팟 위치 조정

사진마다 버튼 위치를 조정하려면 `js/tour.js`에서 `position` 값 수정:

```javascript
hotspots: [
    {
        label: '🚗 주차장길1',
        target: 'parking-road-1',
        position: { x: 30, y: 50 }  // x, y는 백분율 (0~100)
    }
]
```

- `x`: 좌우 위치 (0 = 왼쪽 끝, 100 = 오른쪽 끝)
- `y`: 상하 위치 (0 = 위쪽 끝, 100 = 아래쪽 끝)

## 🎨 커스터마이징

### 새로운 공간 추가

1. **이미지 폴더 생성**
```bash
mkdir images/new-room
```

2. **`js/tour.js`에 공간 추가**
```javascript
'new-room': {
    name: '새로운 방',
    emoji: '🏠',
    photos: ['images/new-room/main.jpg'],
    hotspots: [
        {
            label: '🛋️ 거실',
            target: 'living-room',
            position: { x: 50, y: 50 }
        }
    ]
}
```

3. **연결된 공간의 핫스팟에도 추가**
```javascript
'living-room': {
    // ...
    hotspots: [
        // 기존 핫스팟들...
        {
            label: '🏠 새로운 방',
            target: 'new-room',
            position: { x: 80, y: 60 }
        }
    ]
}
```

4. **`index.html` 메뉴에 버튼 추가**
```html
<li><button class="location-btn" data-location="new-room">🏠 새로운 방</button></li>
```

### 색상 변경

`css/style.css`의 `:root` 변수 수정:

```css
:root {
    --primary-color: #2c3e50;      /* 헤더 메인 색상 */
    --secondary-color: #3498db;    /* 버튼/강조 색상 */
    --accent-color: #e74c3c;       /* 액센트 색상 */
}
```

## ⚠️ 주의사항

1. **사진 파일명**: 기본은 `main.jpg` (또는 `tour.js`에서 경로 수정)
2. **파일 크기**: 너무 크면 로딩이 느려지므로 적절히 압축
3. **로컬 실행**: 브라우저 보안 정책으로 인해 로컬 서버 사용 권장
4. **핫스팟 위치**: 사진을 넣은 후 핫스팟 위치를 조정해야 할 수 있음
5. **동선 일관성**: 연결을 추가/삭제할 때 양방향 모두 수정

## 📞 문제 해결

### 이미지가 로드되지 않는 경우
1. 파일 경로 확인 (대소문자 구분)
2. 파일명이 `main.jpg`인지 확인
3. 브라우저 콘솔(F12)에서 에러 확인
4. 로컬 서버를 사용하고 있는지 확인

### 핫스팟이 잘못된 위치에 있는 경우
1. `js/tour.js`에서 해당 핫스팟의 `position` 값 조정
2. x, y 값을 0~100 사이로 조정
3. 브라우저에서 새로고침하여 확인

### 메뉴가 너무 길어서 스크롤이 안 되는 경우
- 메뉴는 자동으로 스크롤 가능하게 설정되어 있습니다
- 모바일에서는 스와이프로 스크롤 가능

## 💡 팁

### 효율적인 사진 촬영 방법
1. **진입로**: 입구에서 시작해서 안쪽으로 들어가는 순서로 촬영
2. **각 공간**: 중앙에서 주요 연결 지점이 보이게 촬영
3. **여러 각도**: 한 공간에 2~3장씩 다양한 각도로 촬영
4. **조명**: 밝은 시간대에 촬영 (자연광 활용)

### 동선 설계 팁
- 실제 방문객이 이동하는 순서대로 핫스팟 배치
- 중요 공간(거실, 금은별장 입구)은 여러 곳과 연결
- 막다른 공간은 되돌아갈 수 있는 핫스팟 반드시 추가
